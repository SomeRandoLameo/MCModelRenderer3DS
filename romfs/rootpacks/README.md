### This folder should contain minecraft source assets / root resource packs. At least one is required in order for everything to load propperly.

How do i aquire a root Resource pack?

1. Grab it from a Minecraft jar (preferred)
    - Download a minecraft jar using any minecraft launcher
    - copy the jar and rename the file ending from `.jar` to `.zip`
    - open it with your preffered file viewer
    - locate the assets directory
    - copy the assets into this directory
      
2. Download the assets 
    - go to [MCAssets](https://mcasset.cloud)
    - Select your preferred version in the dropdown
    - Download your desired version
    - locate the downloaded directory
    - unzip it
    - copy the assets into this directory
