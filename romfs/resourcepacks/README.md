### This folder should contain additional / external minecraft resource packs. 

How do i use a Resource pack?
- go to [Modrinth | Resource packs]([https://mcasset.cloud](https://modrinth.com/discover/resourcepacks))
- Select your preferred game version in the Game Version list
- Find a pack that you like
- Download your resource pack
- locate the downloaded directory
- unzip it
- copy the assets into this directory
